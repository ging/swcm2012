Ejemplo de Pagina con 2 Columnas

Página formateada con CSS

Esta es la columna principal de esta pagina Web formateada con CSS, que incluye cabecera, 2 columnas y pie de página.
  • Para formatear los bloques se delimitan con
    <div class="TipoBloque"> .. </div>
  • class="TipoBloque" se utiliza para definir cual es el formato de cada tipo de bloque
  • Cada TipoBloque se formatea con un comando CSS.
    TipoBloque {propiedad: valor; ...}
